Mes Travaux Pratiques

TP 1 : SITE-RESTAURANT-BOOTSTRAP

Résumé du TP

Conception et développement d'un site web complet pour un restaurant fictif en utilisant le framework Bootstrap. Le site comprend une page d'accueil attrayante, un menu avec des photos des plats et une section de reservation. Ce projet m'a permis de maîtriser les composants Bootstrap et de créer une interface responsive optimisée pour tous les appareils.

Technologies utilisées

  • HTML
  • CSS
  • Bootstrap
  • JavaScript
  • Responsive Design
